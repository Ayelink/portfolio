# Portfolio Website

个人作品集网站，展示项目、博客和联系方式。

## 特性

- 响应式设计，支持移动端
- 纯静态 HTML/CSS/JS
- GitHub Pages 部署
- 现代化 UI 设计

## 页面

- **首页** - Hero 区域、精选作品、关于我、最新文章、联系方式
- **作品** - 项目作品集展示
- **博客** - 技术文章列表
- **联系** - 联系表单和社交媒体链接

## 部署

本网站使用 GitHub Pages 部署，可通过以下地址访问：
```
https://your-username.github.io/portfolio/
```

## 技术栈

- HTML5
- CSS3 (Flexbox + Grid)
- JavaScript (ES6+)
- GitHub Pages
